Usage — import tokens.css once in your global stylesheet (above Tailwind). Components then read semantic vars: shadcn primitives pick them up automatically; custom layouts use var(--flota-*) or the generated utilities. Switch to a dark theme later by overriding the semantic layer only — primitives stay fixed.
